Условие: найти площадь поверхности тела.

Решение

Воспользуемся заменой координат: $$x = a\cos \varphi$$ $$y = y$$ $$z = a\sin\varphi$$ Искомая площадь равна: $$S = 4a\int\limits_{0}^{a}\int\limits_{-\arcsin{\sqrt{1-\frac{y^2}{a^2}}}}^{\arcsin{\sqrt{1-\frac{y^2}{a^2}}}}y d\varphi$$ $$=8a\int\limits_{0}^{a}y\arcsin\sqrt{1-\frac{y^2}{a^2}}dy = \left[1-\frac{y^2}{a^2}=t, dt = -2\frac{y}{a^2}dy \right]=$$ $$= 4a^3 \int\limits_{0}^{1}\arcsin \sqrt{t}dt = \left[\sqrt{t}=u, dt = 2udu \right] = 8a^3\int\limits_{0}^{1}\arcsin u^2du = $$ $$= 8a^3 \int\limits_{0}^{1}4d\left(u\arcsin u + \sqrt{1-u^2}\right) = 4a^3\pi-8a^3\int\limits_{0}^{1}u\arcsin u du - 2a^3\pi = 2\pi a^3$$

Условие: найти площадь части сферы, заключенной внутри цилиндра.

Решение

Сделаем сферическую замену координат: $$x = a\cos\varphi\sin\psi$$ $$y = a\sin\varphi\cos\psi$$ $$z = a\sin\psi$$ Площадь части сферы, заключенной внутри цилиндра равна: $$S = 8\int\limits_{0}^{}\frac{\pi}{2}\int\limits_{0}^{\arccos\frac{1}{\cos^2\varphi +\frac{a^2}{b^2}\sin^2\varphi}}a^2\sin\psi d\psi d\varphi = $$ $$= 8a^2\int\limits_{0}^{\frac{\pi}{2}}1 - \frac{1}{\cos^2\varphi+\frac{a^2}{b^2}\sin^2\varphi}d\varphi=8a^2\int\limits_{0}^{\frac{\pi}{2}}\frac{\sqrt{a^2-b^2}\sin\varphi d\varphi}{\sqrt{a^2-(a^2-b^2)\cos^2\varphi}} = [t = \cos\varphi] =$$ $$= 8a^2\sqrt{a^2-b^2}\int\limits_0^1\frac{dt}{\sqrt{a^2-(a^2-b^2)t^2}} = 8a^2\arcsin\left(\frac{\sqrt{a^2-b^2}}{a}t\right)\bigg|_0^1 = 8a^2\arcsin\left(\frac{\sqrt{a^2-b^2}}{a}\right)$$

Решение

Сделаем сферическую замену координат: $$x = a\cos\varphi\sin\psi$$ $$y = a\sin\varphi\cos\psi$$ $$z = a\sin\psi$$ $$EG-F^2=a^4\cos^2\psi$$ Ограничение на $\varphi$: $$|\cos\varphi| \leq \cos\psi$$ Поэтому искомая площадь равна: $$S=8\int\limits_0^{\frac{\pi}{2}}\int\limits_0^\varphi a^2\cos\psi d\psi d\varphi = 8a^2\int\limits_0^{\frac{\pi}{2}}\sin\varphi = 8a^2(-\cos\varphi)\bigg|_0^{\frac{\pi}{2}} = 8a^2$$

Условие: найти площадь геликоида.

Решение

$$E=\cos^2\varphi+sin^2\varphi=1$$ $$G = r^2\sin^2\varphi+r^2\cos^2\varphi+h^2=r^2+h^2$$ $$F = -r\cos\varphi \sin\varphi+r\cos\varphi\sin\varphi = 0$$ $$EG-F^2=r^2+h^2$$ Значит, площадь равна: $$S = \int_0^{a}\int_0^{2\pi}\sqrt{r^2+h^2}d\varphi dr = \left[t = ln \left(\frac{a}{h} + \sqrt{\frac{a^2}{h^2}+1}\right) \right] = 2\pi h^2\int\limits_0^{ln \left(\frac{a + \sqrt{a^2+h^2}}{h}\right)}\cosh^2 tdt=$$ $$=\pi h^2 ln \left(\frac{a + \sqrt{a^2+h^2}}{h}\right)+\pi h^2\frac{a}{h}\sqrt{1+\frac{a^2}{h^2}} = \pi h^2 ln \left(\frac{a + \sqrt{a^2+h^2}}{h}\right)+\pi a\sqrt{a^2+h^2}$$

Условие: найти площадь части поверхности тора, ограниченной двумя параллелями $\psi_1, \psi_2$ и двумя меридианами $\varphi_1, \varphi_2$. Чему равна площадь всего тора?

Решение

$$E=(b+a\cos\psi)^2$$ $$G = a^2$$ $$F=-(b+a\cos\psi)\sin\varphi(b-a\cos\psi)\cos\varphi+(b+a\cos\psi)\cos\varphi(b-a\cos\psi)\sin\varphi=0$$ Значит, площадь равна: $$S=a\int\limits_{\varphi_1}^{\varphi_2}\int\limits_{\psi_1}^{\psi_2}(b+a\cos\psi)d\psi d\varphi = a(\varphi_2-\varphi_1)(b(\psi_2-\psi_1)+a(\sin\psi_2-\sin\psi_1))$$ Получаем, что площадь полного тора равна $4\pi^2ab$
